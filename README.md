## Appointment Web Application

An appointment management application that allows users to add, view, search, sort, and delete appointments.

## Features

- **Add Appointment** with fields for first name, last name, date, time, and notes  
- **View appointments** in a structured list  
- **Search functionality** to filter appointments by name or notes  
- **Sorting controls** (first name, last name, Appointment Date, ASC(A to Z), DESC(Z to A)  
- **Delete appointment** with one click  
- **Dynamic toggle form** for showing or hiding the appointment form  
- **Loads initial data** from an external JSON file  
- **Responsive layout** using React Bootstrap  
- **Background image and blur UI effect** in the add-appointment form  
- **Uses React Icons** for visual enhancements  

## Setup Instructions

### 1. Install dependencies
```bash
npm install
```
### 2. Start the development server
```bash
npm start
```
